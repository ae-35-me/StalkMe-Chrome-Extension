## Installation

1.  Clone this repository.
2.  Open Chrome and go to `chrome://extensions`.
3.  Enable **Developer Mode** (top right).
4.  Click **Load unpacked**.
5.  Select the directory containing this extension.

## Usage

1.  Click the extension icon.
2.  View your cookies and highlighted session tokens.
3.  Enter your **Gemini API Key** in the input field.
4.  Click **Analyze My Tabs** to see what the AI thinks of your browsing habits.

## Disclaimer

This extension is for **educational and demonstration purposes only**. It requests extensive permissions (`webRequest`, `cookies`, `tabs`) to demonstrate how much data an extension can access. Use with caution.
